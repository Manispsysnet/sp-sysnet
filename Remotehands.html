SPS Smart Hands Services

Smart Hands services take issue for each colocation facility. To administer you a short plan, here are a number of the services that cover:

  • Provide on-site/off-site technical support
  • Configuration of the firewall
  • Complex cable configurations
  • Placement device management
  • Test devices and fix errors
  • Rack and stacking services
  • Assemble, install and maintain cabling.
  • Interface card removal, installation, and configuration
  • Setting up and Maintaining Backups
  • Install, replace or uninstall , like a router, switch card, disk drive, memory, etc.
  • Take inventory, recording digitally, and serial numbers.
  • Detect and answer threats
  • Moving instrumentation
  • Operating system and server installation
  • Perform an power cycle a router, server, or switch; soft-boot a server.
  • On-site and off-site technical support and troubleshooting
  • Remote server restarts
  • Cloud cross-connect

SPS Remote Hands Services

Remote hands are a progression of on-site, actual IT management and maintenance services given by datacentres and colocation suppliers.

  • Moving or securing network cables
  • Cabling and wiring
  • Network router and switch configuration
  • Server refreshes and reboots
  • Hardware and software replacement or installation
  • Media Management
  • Inventory management and labelling
  • Shipping and receiving requests
  • Verify equipment performance
  • Audits
